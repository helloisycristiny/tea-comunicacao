# TEA Comunicação

Aplicativo interativo desenvolvido para auxiliar a comunicação de crianças com Transtorno do Espectro Autista (TEA).

# Sobre o Projeto

O TEA Comunicação foi desenvolvido com o objetivo de promover acessibilidade e inclusão por meio da tecnologia, oferecendo recursos que auxiliam a comunicação de crianças com TEA em diferentes contextos.

#Publicação Científica

* TEA Comunicação: Um Aplicativo Interativo para Crianças com Transtorno do Espectro Autista

Publicado nos Anais do XVII Seminário de Iniciação Científica, Tecnológica e Inovação (SICTI) do Instituto Federal do Pará.

# Objetivos

- Promover acessibilidade comunicacional;
- Auxiliar crianças com TEA em suas interações;
- Utilizar tecnologia para inclusão social;
- Apoiar familiares, cuidadores e profissionais.

# Tecnologias

- HTML
- CSS
- JavaScript

# Reconhecimentos e Participações

O projeto TEA Comunicação obteve destaque em eventos acadêmicos e de inovação:

- Apresentado no XVII Seminário de Iniciação Científica, Tecnológica e Inovação (SICTI) do IFPA (2025); 
- Apresentado no IFS do Norte durante a programação relacionada à COP30, em Belém (PA).

Essas participações contribuíram para a divulgação de soluções tecnológicas voltadas à inclusão e acessibilidade para pessoas com Transtorno do Espectro Autista (TEA).
